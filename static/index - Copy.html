How I got this site up and running for under a dollar

  1. The domain was registered with Namecheap for $0.98 using their 20K tweets promotional coupon.

    This is the only money I spent on this project. Even without a coupon, a domain can be registered for about $5.00 or less.

  2. The content is hosted on Google App Engine(GAE) free tier @kjp-blog.appspot.com

    GAE is perfect for hosting a small traffic website like this(the only visitors so far are me and my wife). The free tier usage quotas are quire liberal and provides 28 free instance hours which is enough to run the site continously. And even though GAE is meant to host applications written in Python/Java/Go, it works just fine to serve pure static content.

  3. The page is created using Bootswatch - a better looking CSS for twitter Bootstrap.

    Right now there is no CMS or blogging software used, it is all coded by hand - which is fine since there is only one post. I wanted to use WordPress , but the PHP support in GAE is in beta and by invitation only at this time.

  4. Linked appspot app with the domain kesuj.org using a new CNAME entry in DNS. Added the domain in GAE.

    This was the difficult part but this answer in StackOverflow helped.

  5. And now I wait. For visitors. Anyone? Hello...? Chirp..chirp..
